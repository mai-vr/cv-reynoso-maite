# Trabajo práctico N°1 - Curriculum web
Programacion 3 - Tecnicatura Universitaria en Programación (UTN).
Maite Reynoso.

---

Este proyecto es una página web de mi curriculum vtae donde se detalla mi experiencia laboral, los
lenguajes que conozco y los proyectos que he realizado.
Se puede ver accediendo a este link: [CV Web](https://mai-vr.github.io/cv-reynoso-maite/)

## Tecnologías utilizadas
- HTML5 - Maquetación del sitio.
- CSS - Estilos de la página.
- JavaScript - Interacciones como envío del formulario y visualización de información sobre la experiencia laboral.

## Visualización del proyecto
Para poder visualizar el proyecto, se puede acceder al link [CV Web](https://mai-vr.github.io/cv-reynoso-maite/) o clonarlo desde github, para eso se puede utilizar en la terminal local: 
```
git clone https://github.com/mai-vr/cv-reynoso-maite.git
```
Luego, hay que abrir index.html en cualquier navegador. Para esto, es necesario posicionarse sobre el archivo, hacer click derecho, seleccionar la opcion 'Open with' y elegir el navegador.
